# Judges Handout - Instructions

## How to Use the Handout

### Option 1: Print from Browser (Recommended)
1. Open `JUDGES_HANDOUT.html` in any web browser (Chrome, Firefox, Safari, Edge)
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac) to open print dialog
3. Select "Save as PDF" or print directly
4. Choose "A4" paper size
5. Set margins to "Default" or "Minimum"
6. Print or save

### Option 2: Convert to PDF Online
1. Open `JUDGES_HANDOUT.html` in a browser
2. Use browser's "Print to PDF" feature
3. Or use online tools like:
   - HTML to PDF converters
   - Adobe Acrobat
   - Browser extensions

### Option 3: Print Directly
1. Open `JUDGES_HANDOUT.html` in a browser
2. Print on A4 paper (color recommended for best visual impact)
3. Ensure "Background graphics" is enabled in print settings
