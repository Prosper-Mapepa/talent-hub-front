# CMU Talent Hub - Business Plan (One-Page Executive Summary)

## Business Model, Product/Service, and Target Market

**Product:** CMU Talent Hub is a verified, secure platform connecting CMU students with opportunities, peers, and businessesâ€”LinkedIn meets Upwork, built exclusively for CMU students.

**Business Model:**
- **Freemium:** Free basic profiles; Premium subscriptions ($9.99/month students, $49.99/month businesses)
- **Transaction Fees:** 5% commission on successful job placements
- **Featured Listings:** $99/month premium job postings
- **Campus Partnerships:** Revenue sharing with CMU career services

**Target Market:**
- **Primary:** 15,000+ CMU students seeking internships, projects, and career opportunities
- **Secondary:** 1,400+ CMU faculty members and hundreds of recruiting businesses

---

## Technology Integration

**Platform Stack:**
- **Frontend:** Next.js 15 + React 19 responsive web application
- **Mobile:** React Native + Expo cross-platform app (iOS & Android)
- **Backend:** NestJS RESTful API with PostgreSQL database
- **AI/ML:** Intelligent talent matching algorithms, automated job recommendations, smart search with NLP
- **Real-time:** WebSocket integration for instant messaging and notifications
- **Security:** JWT authentication, role-based access control, encrypted communications, CMU email verification
- **Cloud:** Integrated file storage for portfolio media (videos, images, code repositories)

**Technology Advantages:** Scalable microservices architecture, API-first design, mobile-first approach, automated verification system
